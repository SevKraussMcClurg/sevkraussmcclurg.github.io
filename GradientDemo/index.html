Gradient Demo

Welcome to my demo on how to use gradients

Content From css-tricks.com

There are 3 types of gradients: linear, radial, and conic. Here are what they look like and the code for them.

Linear

                        
.linear {
    background-image: 
        linear-gradient(
            #b7f6f7, #ff80ea
        );
}

Radial

                        
.radial {
    background-image: 
        radial-gradient(
            #b7f6f7, #ff80ea
        );
}

Conic

                        
.conic {
    background-image: 
        conic-gradient(
            #b7f6f7, #ff80ea
        );
}

Linear Gradient Options

You can also specify the direction of linear gradients by using terminology like "to top right" or "90deg"

                    
.linear {
    background-image: 
        linear-gradient(
            to top right, #b7f6f7, #ff80ea
        );
}
                    
                
                    
.linear {
    background-image: 
        linear-gradient(
            90deg, #b7f6f7, #ff80ea
        );
}

You can also use more colors, just seperate them by commas

                        
.linear {
    background-image: 
        linear-gradient(
            red, yellow, green, blue
        );
}

Radial Gradient Options

Radial gradients have a gradient that starts at a specific point and radiate out from there. You are able to specify between a circle or ellipse shape (cicles work well for perfect squares and ellipses work well for rectangles). CSS will automatically choose the one that matches the box shape you have but you can also specify it if you wanted.

                        
.radial {
    background-image: 
        radial-gradient(
            circle, #b7f6f7, #ff80ea
        );
}

Similarly to linear gradients, you can specify the starting point in radial gradients

                            
.radial {
    background-image: 
        radial-gradient(
            circle at bottom, #b7f6f7, #ff80ea
        );
}

Also similarly to linear gradients, you can also add more colors in radial gradients

                            
.radial {
    background-image: 
        radial-gradient(
            red, yellow, green, blue
        );
}

Conic Gradient Options

Conic gradients look like a cone when viewed from above. CSS will assume that each color is spaced evenly if you don't specify angles (in degrees or percentages). Here is an example of specified angles.

                            
.conic {
    background-image: 
        conic-gradient(
            #b7f6f7 0 30%, #ff80ea 30% 70%, #ff6a6a 80% 0
        );
}

Yet again, you can use more than 2 colors and if you end with the same color you start with, you are able to remove the harsh line.

                        
.conic {
    background-image: 
        conic-gradient(
            red, orange, yellow, green, blue, magenta, red
        );
}

Repeating Gradients

You can have gradients repeat which can create interesting patterns.

                        
.linear {
    background-image: 
        repeating-linear-gradient(
            45deg, #b7f6f7, #b7f6f7 10px, #ff80ea 10px, #ff80ea 20px
        );
}
                        
                    
                        
.radial {
    background-image: 
        repeating-radial-gradient(
            circle at 0 0, #b7f6f7, #b7f6f7 10px, #ff80ea 10px, #ff80ea 20px
        );
}
                        
                    
                        
.conic {
    background-image: 
        repeating-conic-gradient(
            from 0deg, #b7f6f7 0 30deg, #ff80ea 30deg 60deg
        );
}

Other Patterns

Checkerboard

                        
.checkerboard {
    background-color: #fff;

    background-image: 
        linear-gradient(
            45deg, #333 25%, transparent 25%, transparent 75%, #333 75%, #333
        ),
        
        linear-gradient(
            -45deg, #333 25%, transparent 25%, transparent 75%, #333 75%, #333
        );
}

Chevron

                        
.chevron {
    background-color: #b7f6f7;

    background-size: 50px 50px;

    background: 
        linear-gradient(
            135deg, #ff80ea 25%, transparent 25%
        ) -25px 0,

        linear-gradient(
            225deg, #ff80ea 25%, transparent 25%
        ) -25px 0,

        linear-gradient(
            315deg, #ff80ea 25%, transparent 25%
        ),

        linear-gradient(
            45deg, #ff80ea 25%, transparent 25%
        );
}

Graph Paper

                        
.graphpaper {
    background-color: #eaeaea;

    background-size: 20px 20px;

    background-image: 
        repeating-linear-gradient(
            45deg,rgba(0, 191, 255, 0.5),rgba(0, 191, 255, 0.5) 1px, transparent 1px, transparent 15px
        ),

        repeating-linear-gradient(
            -45deg,rgba(255, 105, 180, 0.5),rgba(255, 105, 180, 0.5) 1px, transparent 1px, transparent 15px
        );
}

Border Gradients

Normal linear gradient
                        
.border-gradient-normal {
    background-image: linear-gradient(45deg, #b7f6f7,#ff80ea);
    border-radius: 16px;
    padding: 10px;
    height: 108.6px;
}
                            
.border-gradient-normal .content {
    background: #fff;
    border-radius: 8px;
    box-shadow: 0 0 15px rgba(0, 0, 0, 0.25);
    padding: 35px;
    text-align: center;
}
Dotted radial gradient
                        
.border-gradient-dotted {
    background-image: radial-gradient(#b7f6f7, 35%, transparent 35%);
    background-size: 8px 8px;
    border-radius: 16px;
    padding: 35px;
    height: 158.6px;
}
                            
.border-gradient-dotted .content {
    background: #fff;
    border-radius: 8px;
    box-shadow: 0 0 15px rgba(0, 0, 0, 0.55);
    padding: 35px;
    text-align: center;
}

Animated Gradient

From Manuel Pinto on CodeOpen

                        
.animated-gradient {
    animation: gradient 8s ease infinite;
    background-image: linear-gradient(135deg, #b7f6f7, #44aaac, #ff93ed, #630053);
    background-size: 400% 400%;
}
                            
@keyframes gradient {
    0% {
        background-position: 50% 0%;
    }
    50% {
        background-position: 100% 50%;
    }
    100% {
        background-position: 50% 0%;
    }
}
